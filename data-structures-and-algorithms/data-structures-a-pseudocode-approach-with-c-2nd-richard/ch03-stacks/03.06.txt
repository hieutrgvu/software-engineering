3.5 If the values of A, B, C, and D are 2, 3, 4, and 5, respectively, manually
calculate the value of the following prefix expressions:

a. + – * A B C D
b. – * A + B C D

********************************************************************************
Resolve by: Hieu Vu
********************************************************************************

a. + (– (* A B) C) D = ((A * B) - C) + D = ((2 * 3) - 4) + 5 = 7
b. – (* A (+ B C)) D = (A * (B + C)) - D = (2 * (3 + 4)) - 5 = 9
