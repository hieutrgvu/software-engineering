6.24 What is the minimum number of levels a ternary tree, that is, a tree with
an outdegree of 3, with 42 nodes can have?

********************************************************************************
Resolve by: Hieu Vu
********************************************************************************

3^0 + 3^1 + 3^2 + ... + 3^level = max number of nodes
-> minimum number of levels (height) of 42 nodes is 5.
