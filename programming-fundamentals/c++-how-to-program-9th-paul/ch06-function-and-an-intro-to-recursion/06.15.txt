6.15 (Short Answer Questions) Answer each of the following questions:
a) What does it mean to choose numbers “at random?”
b) Why is the rand function useful for simulating games of chance?
c) Why would you randomize a program by using srand? Under what circumstances
is it desirable not to randomize?
d) Why is it often necessary to scale or shift the values produced by rand?
e) Why is computerized simulation of real-world situations a useful technique?

********************************************************************************
Resolved by: Hieu Vu
********************************************************************************

a) Every number in a specific range has an equal chance to be chosen.
b) Because it generate a sequence of pseudo-random number.
c) rand will generate the same sequence if not using srand. Under debugging 
circumstance, it is desirable not to randomize.
d) To produce random values in a specific range.
e) Because random data can be analysed and enable more accurate predictions
of random events.
